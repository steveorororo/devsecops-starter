# Threat Model Lite (Chapter 1)

## Assets
- Service availability
- Integrity of API responses
- Future sensitive data

## Key Risks (STRIDE-style)
- Spoofing: unauthenticated access
- Tampering: modified API responses
- DoS: request flooding
- Info Disclosure: leaked secrets/configs

## Mitigations
- CI security scanning
- Secrets scanning
- Dependency scanning
